# Seguimiento-Avance

La visualización elaborada con el código tiene el objetivo de mostrar el porcentaje de avance de los 32 estados de la República en relación con el cumplimiento de un Proyecto determinado. 

Se agregó una línea de referencia que corresponde al porcentaje de avance que deberían tener todos los estados a la fecha. Conforme pase el tiempo, esta se recorrerá a la derecha al igual que el avance de cada estado. 

Se utilizaron tonos morados y rosas ya que corresponden a los colores institucionales.
